# Nokair Platform Monorepo

This monorepo contains:

- `apps/nokair-web` – Nok Air public website (Next.js + shadcn/ui)
- `apps/nokair-web-backend` – Website backend (Go + Postgres)
- `apps/nokair-cms-admin` – CMS admin panel (Next.js + shadcn/ui)
- `apps/nokair-cms-backend` – CMS backend (Go + Postgres)
